Under the Account menu you may choose login, feedback or logout.

Login

Some parts of the app, such as 'Analyze' requires login and will automatically log you in if you haven't. But you may also manually login using the Account menu.

Logout

You don't need to logout if you're the only one has access to the device.

Note: PI360 does not authenticate users, instead it uses social authentication to log user in. If the same PI360 has authenticated to multiple social sites -- this could happen, when a user has logged in using Facebook then analyzes someone at Twitter. PI360 then will map both Facebook and Twitter account to the same PI360 account.

Feedbacks

For assistance, bug report, feature request or just general feedbacks, please use the Feedbacks menu.
